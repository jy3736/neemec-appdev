Step-by-step Flutter Installation

Step 1 : Download All Required/Optional SDK and Tools

Step 2 : Update your path

  • From the Start search bar, enter edit (編輯) or and select Edit environment variables for your account.
  • Under Environment variables looking for an entry called Path:
    • Add the full path to flutter\bin
      • eg. C:\tools\flutter\bin

Step 3 : Run flutter doctor

flutter doctor

Step 4 : Install Android Studio

  1. Download and install Android Studio.
  1. This installs the following packages. They are required by Flutter when developing for Android.
    1. Android SDK
    1. Android SDK Command-line Tools
    1. Android SDK Build-Tools
  1. Run flutter doctor to confirm that Flutter has located your installation of Android Studio.

Step 5 : Setup Your Android Device (Mobile Phone)

  1. Enable Developer options and USB debugging on your device.
  1. Windows-only: Install the Google USB Driver.
  1. Using a USB cable, plug your phone into your computer. If prompted on your device, authorize your computer to access your device.
  1. In the terminal, run the flutter devices command to verify that Flutter recognizes your connected Android device.

Step 6 : Setup the Android Emulator

  1. Enable VM acceleration on your machine.
  1. Launch Android Studio, click the AVD Manager icon, and select Create Virtual Device.

Step 7 : Agree to Android Licenses

flutter doctor --android-licenses

Update Flutter to the latest version

flutter upgrade
